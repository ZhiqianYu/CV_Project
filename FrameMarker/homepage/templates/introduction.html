Introduction

Frame Annotator is designed for capturing and annotating information for each frame of a video.

Upload

The video upload process consists of three steps:

  1. Users need to register for a personal account and log in.
  2. After logging in, return to the upload video interface to upload your video. Note: there are no restrictions on the video file format; All uploaded videos are automatically converted to the .mp4 format in the background.
  3. Upon successful upload, an information box will confirm the successful upload. Finally, you can click on the 'videos' interface to proceed with editing.

Videos

Video List Display:

  • Displays a list of uploaded videos, including those uploaded by the user and others.
  • Each video provides detailed information such as the file name, uploader, and upload time.
  • Features a filtering option for quick access to desired videos based on criteria such as file name, uploader, and whether the video has been annotated.

Frame Annotation:

Located in the top left of the video, there is a 'Not Annotated' label that can be clicked to enter the frame annotation interface.

The annotation page consists of three parts:

  1. Play Video
  2. Frame Generation: Click the 'Frame Generation' button to obtain all frames of the video. Note: This process may take some time! Once all frames are loaded, a notification will appear, and the interface needs to be refreshed.
  3. Image Selection: The right side of the video player displays all images representing the total duration of the video. By randomly selecting an image, you can obtain four frames within 1 second below the video player. The selected image is also displayed in the player.

Users can annotate the selected images, indicating attributes such as 'good' or 'bad.' The annotations are automatically saved.

Exports

The "Exports" interface serves as a dedicated export page with the primary purpose of facilitating the filtering of frame annotation information based on specific users and videos. This feature enables users to tailor their exports according to their criteria, enhancing flexibility in data retrieval.

One key functionality of this interface is the ability to export the filtered frame annotation data from the database. The exported data is formatted in CSV (Comma-Separated Values) file format, ensuring compatibility and ease of use for further analysis or integration with other tools.

The exported information encompasses several crucial details associated with each frame annotation. This includes the video file name, providing a reference to the source material. Additionally, details about the uploader, upload time, frame type, frame number, and frame rating are included, offering a comprehensive set of information for analytical or archival purposes. The "Exports" interface thus streamlines the process of obtaining specific frame annotation data based on user-defined parameters.
